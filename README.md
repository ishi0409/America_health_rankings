# 概要
BigQueryの一般公開データであるamerica_haelth_rankigsを用いてデータの抽出及び可視化を行った。  

# 目的
BigQuery及びBIツールの使い方を実際にデータを用いて行い身に着けるため。  
また、膨大なデータから知見を得る練習を行うため。  

# 本分析のポイント
本分析はamerica_health_rankingsのデータにおいて、food Insecurity（食料不安)を抱えているデータをもとに人種や学歴といった指標であるsubpopulationカラムでグループ化を行い、グループ別の欠測値数及び、食料不安値の比較を行った。  

# 本分析からわかったこと
本分析から得られた知見としては人種が食料不安値及び、欠測値数に影響を及ぼしていることが分かった。  
特にネイティブアメリカン、黒人、マルチレイシャルグループが食料不安値及び欠測値の値が高い傾向があった。  

# 今後の展望
本分析から人種が食料不安に関係を持っている可能性があることが分かった。
したがって、人種と食料不安の関連を含んだデータを用いて分析を行うことでより詳しい知見が得られると考える。  
また、本データではグループ別の具体的な人数が不明なため人数分布なども含んだデータを用いることで同様に新たな知見が得られると考えた。  

# 使用したツール
BigQuery,Lookerstudio

# Looker studio 共有リンク
[https://lookerstudio.google.com/s/reXFgT7ZoaA]

棒グラフが２枚、その横に数値が表示されています。  
特定のカラムをクリックすることでそのグラフの値がそれぞれのグラフにおいて表示されます。  
上が欠測値数のグラフで、下が食料不安値のグラフとなっています。  
右上のリセットを押すことで初期状態に戻ります。
